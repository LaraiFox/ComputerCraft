Module TurtleUtils.lua

TurtleUtils is a lua file designed to add extra utility functions to ComputerCraft's Turtle API and to slightly modify some of the functions that already exist.

Some examples of functions that TurtleUtils adds include turning to face a specified direction, moving to a specified location, etc.

Turtle movement and rotation functions work the same way as before except you can now optionally specify how many times you want the function to repeat rather than calling it multiple times. These functions now also automatically track the turtle's position and the direction it's facing.

Global overriden turtle functions

turtle.turnLeft (count)
Turns the turtle 90 degrees to the left and decrements turtle_Direction the number of times specified. If count is nil then count defaults to 1. If count is negative the turtle will turn right instead.

Parameters:

  • count number of times the turtle will turn left
turtle.turnRight (count)
Turns the turtle 90 degrees to the right and increments turtle_Direction the number of times specified. If count is nil then count defaults to 1. If count is negative the turtle will turn left instead.

Parameters:

  • count number of times the turtle will turn right
turtle.down (count)
Moves the turtle down by one block a specified number of times and decrements turtle_Position's y value. If count is nil then count defaults to 1. If count is negative the turtle will move up instead.

Parameters:

  • count number of times the turtle will move down
turtle.up (count)
Moves the turtle up by one block a specified number of times and increments turtle_Position's y value. If count is nil then count defaults to 1. If count is negative the turtle will move down instead.

Parameters:

  • count number of times the turtle will move up
turtle.backwards (count)
Moves the turtle backwards by one block a specified number of times and changes turtle_Position based on the current turtle_Direction. If count is nil then count defaults to 1. If count is negative the turtle will move forwards instead.

Parameters:

  • count number of times the turtle will move backwards
turtle.forwards (count)
Moves the turtle forwards by one block a specified number of times and changes turtle_Position based on the current turtle_Direction. If count is nil then count defaults to 1. If count is negative the turtle will move backwards instead.

Parameters:

  • count number of times the turtle will move forwards

turtle.pathfind (location, map, origin)
Attempts to find a path to the specified location using the map parameter as a guide.

Parameters:

  • location the destination coordinates
  • map a map of the area. The map uses 0 to represent empty blocks and 1 for solid blocks.
  • origin the offset of the map from the turtle's starting location.

Returns:

    a table of turtle move commands
turtle.moveTowards (location, direction, order)
Moves the turtle to the specified location parameter in straight lines along the axes in a certain order. When the turtle finishes moving it faces the direction parameter.

Parameters:

  • location the destination coordinates
  • direction the final look direction after moving
  • order the order of axes the turtle will move along

turtle.turnTowards (direction)
Turns the turtle to face the direction parameter with as few calls to the turn functions as posible. If the direction spectified is not between 0 and 3 inclusive the program exits and prints an error.

Parameters:

  • direction the direction the turtle should face
turtle.returnToStart (direction, order)
Moves the turtle back to its starting location in straight lines along the axes in a certain order. When the turtle finishes moving it faces the direction parameter.

Parameters:

  • direction the final look direction after moving
  • order the order of axes the turtle will move along
